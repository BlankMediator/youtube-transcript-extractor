# Chrome Web Store Listing Draft

## Name

YouTube Transcript Extractor

## Short Description

Extract visible YouTube transcripts, view them on-page, copy them, or download TXT/JSON.

## Detailed Description

YouTube Transcript Extractor adds a simple on-page panel to YouTube video pages. Open the transcript on YouTube, click the extension, and extract the visible transcript rows into a clean text area.

Features:

- View transcript text directly on the YouTube page.
- Toggle timestamps on or off.
- Copy the transcript to the clipboard.
- Download TXT for reading or notes.
- Download JSON for structured workflows.

The extension works locally in your browser. It does not send transcript text to any external server.

Note: The extension can only extract transcript rows that YouTube renders for the current video and browser session. If YouTube does not show a transcript, the extension cannot create one.

## Category

Productivity

## Single Purpose

Extract transcript rows already rendered by YouTube and let the user view, copy, or download them locally.

## Privacy Disclosure Notes

This extension does not collect or transmit user data. Transcript text is read from the current YouTube page and remains local unless the user copies or downloads it.

## Reviewer Test Instructions

1. Install the extension.
2. Open a YouTube video that has an available transcript.
3. Click the extension toolbar icon.
4. Click `Open Transcript` if the transcript panel is not already open.
5. Click `Extract`.
6. Verify transcript text appears in the extension panel.
7. Test timestamp toggle, Copy, TXT, and JSON buttons.
