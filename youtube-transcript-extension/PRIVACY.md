# Privacy Policy

YouTube Transcript Extractor does not collect, sell, transmit, or share personal data.

The extension runs locally in the user's browser on YouTube video pages. It reads transcript rows that YouTube has already rendered in the page and displays them in an on-page panel. Copying or downloading transcript text is initiated by the user and stays on the user's device.

The extension does not use analytics, remote servers, tracking pixels, advertising identifiers, or third-party APIs.

Permissions are limited to:

- `storage`: remembers the user's preferred floating button position in the browser.
- YouTube watch page content script access: reads transcript rows rendered on YouTube video pages.
