# Privacy Policy

YouTube Transcript Extractor does not collect, sell, transmit, or share personal data.

The extension runs locally in the user's browser on the active YouTube video page after the user clicks the extension button. It reads transcript rows that YouTube has already rendered in the page and displays them in an on-page panel. Copying or downloading transcript text is initiated by the user and stays on the user's device.

The extension does not use analytics, remote servers, tracking pixels, advertising identifiers, or third-party APIs.

Permissions are limited to:

- `activeTab`: allows the extension to run on the current YouTube video tab only after the user clicks the toolbar button.
- `scripting`: injects the extension's packaged script and CSS into the current YouTube video tab after the user clicks the toolbar button.
